Setup Guide

            

                

                    A
                    

                        
Clientless (Recommended)

                        
    
                            
  1. Download "Full Package" above and extract to any folder
    2. 
                            
  2. Run setup_agent.bat
    3. 
                            
  3. Enter the server URL when prompted (e.g. http://192.168.1.100:8000)
    4.

  4. Scrcpy will auto-download from server on first run
