# Screenshots

Place application screenshots in this folder for use in the main project README.

Suggested filenames:

- `login.png`
- `dashboard.png`
- `devices.png`
- `remote.png`
- `reports.png`
- `automation.png`
- `settings.png`
- `scrcpy-agent.png`

### Devices
![Devices](docs/screenshots/devices.png)

### Automation
